**Vertex coloring** is a technique that assigns a color to each vertex of a geometry. Most three.js materials support vertex coloring.

<Tabs items={tabItems} />

## Usage

{/* prettier-ignore-start */}

<Steps>
1. Assign a **color** attribute to the buffer geometry.

    ```ts
    geometry.setAttribute("color", colorAttribute);
    ```

    <Aside>
        The buffer attribute's name **must** be "color" in order for vertex coloring
        to take effect.
    </Aside>

2. Set the **vertexColors** property of the material to **true**.

    ```ts
    material.vertexColors = true;
    ```
</Steps>
